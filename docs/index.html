FFmpeg Subtitle Tools

n8.0.1-pgs.0 21 patches on FFmpeg 8.0.1. Pre-built static binaries for 6 platforms.

FFmpeg can transcode between virtually any audio and video format. Subtitles are the exception.

"Subtitle encoding currently only possible from text to text or bitmap to bitmap"

That's #3819, open since 2014. These builds fix it — 88 new conversion pairs across both directions, plus a PGS encoder (#6843) and direct RGBA-to-GIF encoding.

Each target has two variants: minimal (ffmpeg + ffprobe) and -eng (adds English OCR data). Other languages: download .traineddata from the tessdata release into a tessdata/ directory next to the binary.

Usage

# SRT/ASS/WebVTT to Blu-ray PGS
ffmpeg -i input.mkv -c:s pgssub -c:v copy -c:a copy output.mkv
ffmpeg -i subtitles.srt -s 1920x1080 output.sup

# PGS/DVD/DVB bitmap to SRT via OCR
ffmpeg -i input.mkv -c:s srt output.srt
ffmpeg -i input.mkv -c:s srt -sub_ocr_lang jpn output.srt

# RGBA to GIF without filter pipeline
ffmpeg -i input.mp4 -c:v gif out.gif

Any text subtitle decoder to any bitmap encoder, and vice versa. Conversion is automatic — just set the output codec.

Animation

ASS effects like \fad, \move, and colour transforms are rendered frame-by-frame and classified automatically. Fades use palette-only Display Sets (~1 KB each) so smooth animation fits within the Blu-ray decoder model. No format-specific tag parsing required.

Release notes

First stable release based on an official FFmpeg release branch. Previous builds tracked master, which changes daily. The n8.0.1-pgs.0 tag adds only our 21 subtitle patches on top of unmodified FFmpeg 8.0.1. All upstream FATE tests pass.

  • GIF RGBA fix: the palette mapper sorts entries internally for KD-tree construction, but the encoder was writing indices against the original unsorted palette — producing blank or garbled output. Fixed by reading the sorted palette back from the mapper.
  • FATE source fix: tests/ref/fate/source updated to include av_clip references from quantize.c, squashed into the originating commit on both branches.
